# Smart Traffic Vehicle Counter

This project uses a web-based UI and a Python backend (YOLOv8) to count vehicles
using region-based detection.

---

## Requirements

- Python 3.9+
- pip
- (Optional) GPU + CUDA for faster inference

---

## Setup

### 1. Create virtual environment
```bash
python -m venv .venv
```
### 2. Activate virtual environment
```bash
.\.venv\Scripts\activate
```
### 3. Install Dependencies
```bash
pip install -r requirements.txt
```
### 4. Run the program
Run the program by clicking the play button on vscode
